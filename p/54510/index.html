    • 不同下山点会得到不同下山路径，即局部最优解。

  • 梯度算法介绍

    一定要同时更新，右边算法错误在于θ0更新值被用于 temp1，应该用旧值。

    • 梯度下降算法寻找最小代价过程

      • 如果斜率 == (d(J(θ))/dθ)θ > 0 ，那么当前θ - α*斜率，就是使得θ减小，逼近函数底端
      • 如果斜率 == (d(J(θ))/dθ)θ > 0 ，那么当前θ - α*斜率，就是使得θ增大，逼近函数底端

      • 如果已经在局部最低点

        此时，斜率为0，θ不会被更新，达到局部最优。

      • 自动减小θ变化程度

        原因是因为，越逼近最低点，斜率会自动变小，α*斜率变小—> θ变化程度降低。、
